Trading Simulation
This project is a trading simulation tool that allows users to test different trading strategies using historical stock price data. The simulation uses a simple moving average crossover strategy to generate buy and sell signals, and users can adjust the length of the short and long moving averages to test different variations of the strategy.

The simulation also includes a backtesting feature that allows users to evaluate the performance of their strategy on historical data.

Uploading Data
To upload your own data, go to http://127.0.0.1:5000/upload and select a CSV file containing historical stock price data. The file should have the following columns: date, open, high, low, close, volume.

Testing Strategies
To test a trading strategy, go to http://127.0.0.1:5000/ and adjust the length of the short and long moving averages using the sliders. The chart will update to show the buy and sell signals generated by the strategy.

Backtesting
To backtest a trading strategy, go to http://127.0.0.1:5000/backtest and select the CSV file containing historical stock price data. The simulation will generate buy and sell signals based on the selected moving average lengths, and the backtesting results will be displayed on the page.
